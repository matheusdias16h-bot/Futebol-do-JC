# Futebol do JC

Site para organizar a lista de jogadores para a resenha de futebol.

## Funcionalidades

- Cadastro de jogador (linha/goleiro)
- Lista dinâmica em tempo real usando Firebase
- Limite de jogadores: 18 jogadores de linha ou 15 + 2 goleiros
- Reset automático baseado na data do jogo

## Como usar

1. Clone este repositório
2. Configure seu Firebase e substitua `firebaseConfig` em `js/app.js`
3. Abra `index.html` ou publique no GitHub Pages
